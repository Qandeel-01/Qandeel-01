I'm Qandeel — a developer from Pakistan working at the intersection of computer vision, geospatial intelligence, and full-stack web development.  
I build tools that turn complex data into things you can see, interact with, and act on.  
Recent: a district risk dashboard for Southern Punjab, a deep-learning building footprint extractor from satellite imagery, and a weather-aware fashion app.  
Currently exploring **Neural Radiance Fields (NeRF)** for scene change detection.
